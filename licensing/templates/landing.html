Dúvidas Frequentes

Perguntas frequentes

O dono do grupo precisa autorizar o uso do monitor? ▾
Não. O MilhasUP Monitor funciona de forma 100% externa ao grupo. Ele utiliza a API oficial do Telegram (MTProto) com as suas próprias credenciais de usuário, exatamente como você faria ao ler o grupo manualmente pelo app. Não há necessidade de qualquer autorização, convite especial ou comunicação com o administrador do grupo.
Precisa adicionar um BOT dentro do grupo? ▾
Absolutamente não. Essa é uma das maiores vantagens do MilhasUP Monitor. Nenhum bot precisa estar presente no grupo. O software usa a mesma API que o aplicativo Telegram usa no seu celular — funciona como se você estivesse lendo o grupo manualmente, só que de forma automática e instantânea.
Como o monitor consegue ler o grupo sem ser membro especial? ▾
O Telegram permite que qualquer usuário membro de um grupo leia as mensagens através da API oficial (MTProto). O MilhasUP Monitor usa sua conta do Telegram para isso — a mesma conta que você já usa normalmente. Você só precisa já ser membro do grupo que quer monitorar, assim como qualquer participante comum.
O Telegram vai banir minha conta? ▾
O monitor usa a API oficial do Telegram (MTProto), a mesma que o próprio aplicativo usa. Não é um scraper ou bot não-oficial. O uso é dentro dos limites da plataforma. A funcionalidade de resposta automática segue limites de velocidade para evitar spam.
Funciona em Mac ou Linux? ▾
Atualmente o MilhasUP Monitor é disponibilizado como aplicativo nativo para Windows 10 e 11. Uma versão para Mac está em planejamento. Se você usar um computador Windows para seu balcão, está pronto para usar.
Precisa deixar o computador ligado? ▾
Sim, o computador precisa estar ligado e com acesso à internet para o monitor funcionar. O software pode ser configurado para iniciar automaticamente com o Windows, então basta não desligar o computador quando sair do balcão. Muitos usuários deixam um computador dedicado para isso.
Quanto tempo leva para configurar? ▾
O assistente guiado de configuração leva entre 5 e 10 minutos para a primeira configuração. Cada etapa tem instruções detalhadas e didáticas. A parte que mais demora é criar as credenciais da API do Telegram (em my.telegram.org), que são gratuitas e levam cerca de 2 minutos.
Como recebo a chave de licença após o pagamento? ▾
Após a confirmação do pagamento, a chave de licença é enviada automaticamente para o e-mail cadastrado na Cakto. O processo é imediato. Com a chave em mãos, basta inserir no aplicativo e o monitor estará ativo.
Posso cancelar quando quiser? ▾
Sim. Não há fidelidade ou multa por cancelamento. Basta cancelar a assinatura na plataforma Cakto antes da próxima cobrança. O acesso permanece ativo até o final do período pago.
